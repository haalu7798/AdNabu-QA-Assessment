# AdNabu-QA-Assessment

Quality Assurance Engineer Assignment | AdNabu

Project Description:
This repository contains the manual test cases and automation script for the AdNabu Shopify Store.
Task 1: Test Design (Manual)
I have designed 6 high-quality test cases covering:
- Product Search (Positive, Negative, Edge Case)
- Add to Cart (Positive, Negative, Edge Case)

Task 2: Test Automation
The automation script covers the complete flow: 
1. Handling store password.
2. Searching for a product.
3. Adding the product to the cart.
4. Validation of the result.

How to Run:
1. Clone the repository.
2. Install Selenium: `pip install selenium`
3. Run the script: `python adnaba_test.py`

Test Report:
- Environment:Chrome Browser
- Status: PASS
- Result:Product "Board" searched and added to cart successfully.
